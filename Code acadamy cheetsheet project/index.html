Css Text Cheatsheet

-
Css Property Accepted Values Description
font-family fontname or "font name" Any font named, wrapped in double quotes if there is a space in the name. A comma and space can be used to add back up fonts.
font-size px, %, em(s) px (pixels) is absolute size. % and em are relative to the text size of parent element. Default web browser text size is 16px.
font-weight bold, normal, 100-900 400 is default font-weight. 700 signifies bold, 300 light with the greater and lesser number being extremes.
font-style normal, italic, oblique italic is stylized version of a font based on calligraphy. Oblique is putting the normal version on a angle. Can be used for fonts that don't have a italic version.
text-transform uppercase, lowercase, capitalize Coverts text to all caps, all lowercase or caps the first letter of each word respectively.
line-height em(s) Adjusts the space above text (leading) for readability. Best done in em so it's relative to font-size. 1.4-1.5 is a good starter setting.
letter-spacing, word-spacing em(s) Adjust the space between letters (kerning) or words. Sometimes helpful when a heading is in all caps to increase kerning. Default word spacing is .25em(s).
text-align left, right, center, justify Aligns text in the box based on value. Justify means evey line in a paragraph will take up the full width of the box.
font-style italic Causes text to apper italic.
